# PlaqueTracker iOS App

Kid-friendly iPhone app that connects to PlaqueTracker Arduino device over BLE (primary) and Wi-Fi (optional).

## Front welcome image location

To show a JPEG on the first "Enter App" screen, save your file at:

`PlaqueTracker/PlaqueTracker/Assets.xcassets/WelcomeFront.imageset/welcome-front.jpeg`

The asset name used in code is `WelcomeFront`.
